fcv.genetica.theme Installation
-------------------------------

To install fcv.genetica.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``fcv.genetica.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        fcv.genetica.theme
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
